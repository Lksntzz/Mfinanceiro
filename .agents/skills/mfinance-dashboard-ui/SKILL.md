# MFinanceiro Dashboard UI

Esta skill cuida da camada visual da dashboard.

## Escopo
- tabs
- painéis
- cards
- layout full screen
- spacing
- hierarquia visual
- visão geral
- detalhes
- insights
- histórico
- base financeira

## Regras obrigatórias
- não alterar regra de negócio sem necessidade
- não mover cálculos para arquivos visuais
- respeitar HTML atual
- preservar IDs, classes e seletores usados pelo JS
- evitar duplicidade de controles
- evitar scroll desnecessário
- priorizar leitura rápida e decisão financeira

## Direção visual
Transformar a dashboard em painel de decisão:
- foco em saldo disponível
- limite diário
- dias restantes
- gasto do dia
- alertas claros
- insights acionáveis

## Ao editar UI
- manter consistência visual
- reduzir poluição
- ampliar área útil de gráficos quando necessário
- não adicionar efeitos só por estética

## Saída ideal
- alteração visual objetiva
- sem quebrar tabs
- sem quebrar bindings do dashboard.js
